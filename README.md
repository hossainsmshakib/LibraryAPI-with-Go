# Library Management API (Go & Gin)

This project implements a simple RESTful API for managing a library catalog using Go and the Gin framework. It provides endpoints for retrieving book information, adding new books, and managing book availability (checkouts and returns).

## Introduction

This API is a basic example of a library management system. It demonstrates core concepts of building RESTful APIs with Go and Gin, including routing, handling HTTP requests, working with JSON data, and basic error handling.

## Features

- Retrieve a list of all books.
- Retrieve a specific book by ID.
- Add a new book to the catalog.
- Check out a book (decrement quantity).
- Return a book (increment quantity).

## Getting Started

### Prerequisites

- Go 1.18 or higher
- Git (optional, for cloning the repository)

### Installation

1.  Clone the repository (optional):

    ```bash
    git clone [https://github.com/hossainsmshakib/LibraryAPI-with-Go](https://www.google.com/search?q=https://github.com/hossainsmshakib/LibraryAPI-with-Go)  # Replace with your repo URL
    cd libraryapi
    ```

2.  Initialize Go modules:

    ```bash
    go mod init [github.com/yourusername/libraryapi](https://www.google.com/search?q=https://github.com/yourusername/libraryapi) # Replace with your module path
    ```

3.  Get the Gin framework:

    ```bash
    go get [github.com/gin-gonic/gin](https://github.com/gin-gonic/gin)
    ```

## Usage

1.  Run the API:

    ```bash
    go run main.go
    ```

    The API will be available at `http://localhost:8080`.
